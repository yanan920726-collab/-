# 小说漫画下载台

一个本地运行的网页内容下载工具，用来分析目录、预览正文或图片，并把小说/漫画保存到本机。

## 功能

- 支持多链接输入
- 支持小说正文、漫画图片的自动识别
- 支持目录结构预览
- 支持章节正文和漫画图片预览
- 支持断点续下，已保存的章节和图片会自动跳过
- 支持安全并行下载
- 支持一键打开下载目录

## 使用方式

1. 双击 `双击打开小说下载器.command`
2. 等待浏览器自动打开页面
3. 粘贴小说目录、分类页、漫画页等链接
4. 点击“先看预览”
5. 确认目录和内容没问题后，点击下载

如果 macOS 第一次提示权限或阻止运行，可以右键该文件后选择“打开”。

## 文件说明

- `index.html`：前端页面
- `novel_web_app.py`：本地 Web 服务和 API
- `novel_downloader.py`：抓取、识别和下载逻辑
- `requirements.txt`：Python 依赖
- `start_novel_downloader.command`：启动脚本
- `双击打开小说下载器.command`：推荐双击入口

## 输出结构

小说：

```text
下载的小说/
  书名/
    manifest.json
    书名.txt
    chapters/
      0001_第一章.txt
      0002_第二章.txt
```

漫画：

```text
下载的小说/
  漫画/
    漫画名/
    manifest.json
    chapters/
      0001_第1话/
        0001.jpg
        0002.jpg
```

## 命令行版

也可以直接用命令行下载：

```bash
python3 novel_downloader.py "https://example.com/book/123/"
```

## 说明

- 只处理用户提供的 http/https 链接。
- 某些网站会因为登录、验证码、403/banip 或加密签名拒绝脚本访问，这类限制不会被绕过。
- 如果某个站点结构特殊，需要按站点单独适配。
